===== PAGE 121 =====
Magnetic susceptibility, c
• The tendency of a material to magnetize when exposed 
to a magnetic field
• Interaction of electrons with the magnetic field
• Variations in c cause intrinsic, static magnetic field 
gradients in parts of the brain
• Air in paranasal sinuses adds local heterogeneity to c
• Gradients established by variations in c act to de-phase 
the spins just like the imaging gradients, except that we 
can’t turn the intrinsic gradients off!

coverage)

===== PAGE 163 =====
GRAPPA scheme (R=2)
ACS1 … ACS2 … EPI1 … EPI2 … EPI3 ..............… EPIN
•
The ACS make up a reference data set
•
The omitted k-space lines from each accelerated (under-
sampled) EPI volume are computed using the complete ACS set
•
A fit is performed for each receive coil element separately, 
because each coil element (or loop) receives signal from a 
different part of the head
•
Individual coil images are combined to produce the final image


===== PAGE 164 =====
GRAPPA motion sensitivity
• ACS used to reconstruct all the under-sampled 
time series
• Motion during the ACS affects all data!!!
• Motion after the ACS generates a mismatch 
between the ACS and the current EPI volume
• If subject position changes yet subject is now 
motionless, a mismatch may remain!

Tactics for diagnosis
• Assess temporal stability
• Acquire a short retest
• Make a brief list of possible explanations
• Develop a most likely hypothesis
– Does the problem exist in a different type of scan?
– Can you make the problem worse?
– Consider removing subject & setting up again
– Consider running a short QC test
