• Recall that the Fourier transform of a sinc 
function is a square (or top hat) and vice versa
• Instead of using an on-off (square) RF pulse 
for excitation, if we use a sinc-shaped RF 
pulse then we can select a square “notch” of 
frequencies centered about the frequency of 
the RF pulse.

===== PAGE 46 =====
If the sinc-modulated RF pulse is 
played out while a gradient is on along 
z, the notch of frequencies Δn 
corresponds to a spatial notch of width 
Dz. This is a slice, of thickness Dz.


===== PAGE 47 =====
Slice thickness:
• To change slice thickness we 
can change Gz. If we double Gz 
(bottom) we halve the slice 
thickness.
• Alternatively, we could change 
the RF pulse length. Since time 
and frequency are inverses, to 
halve the slice thickness we 
would need to double the 
duration of the sinc-shaped RF 
pulse. Hence, we control Gz if 
we can, and keep the RF pulse 
duration fixed.

===== PAGE 88 =====
The three “classic” EPI artifacts:
• Ghosting - The appearance of (hopefully) faint replica 
images in the phase encoding dimension. 
• Distortion - Misplacement of voxels along the phase 
encoding dimension.
• Dropout – Regions of signal loss arising from local 
dephasing gradients.
